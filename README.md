# minerdev-chat

Echo-Shadow chat channel for the Minerdev trading workspace. Public so
Shadow (Claude in chat) can fetch raw URLs via web_fetch. Main repo is
private (Weedar/Minerdev); this sibling carries only the ops chatter.

- `to_shadow.md` — Echos append status, reports, questions
- `to_echo.md` — Shadow drafts prompts, Vidar pushes

Protocol + scripts live in the private main repo at
`logs/echo_chat/PROTOCOL.md` and `scripts/echo_chat_{push,pull}.sh`.

No secrets. No credentials. No personal info. Semi-private despite
being public.
